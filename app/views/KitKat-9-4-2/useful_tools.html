Useful tools

18/19 ILR Learner Entry Tool (V1)

The ILR Learner Entry Tool (LET) is an application which enables users to create 2018 to 2019 Individualised Learner Record (ILR) data in XML file format manually using a data entry interface.

Funding information system (FIS)

FIS can be used by data providers to do the following:

  • validate Individualised Learner Record (ILR) data
  • calculate funding and derived variables
  • create a range of reports based on a set of ILR data.

Learning aims

Learning Aim Reference Service (LARS) offers a web based search facility. It allows users to search by most commonly used fields for Qualifications, Units, Apprenticeship Frameworks and Apprenticeship Standards and their associated validity and funding details.
